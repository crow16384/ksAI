# The `ks_capsule_store` Class

A `ks_capsule_store` is a named registry of ks_capsule objects produced
by
[`as_capsules()`](https://crow16384.github.io/ksAI/reference/as_capsules.md),
with study-level metadata for persistence and retrieval.

## Usage

``` r
is_ks_capsule_store(x)
```

## Arguments

- x:

  An object.

## Value

`TRUE` if `x` is a `ks_capsule_store`, otherwise `FALSE`.
