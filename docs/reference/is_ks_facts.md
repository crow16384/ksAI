# The `ks_facts` Class

A compact, retrieval-capable representation of one
[ks_context](https://crow16384.github.io/ksAI/reference/is_ks_context.md)'s
rows. Build with
[`as_facts()`](https://crow16384.github.io/ksAI/reference/as_facts.md),
subset with
[`retrieve()`](https://crow16384.github.io/ksAI/reference/retrieve.md),
render with
[`as_compact()`](https://crow16384.github.io/ksAI/reference/as_compact.md).

## Usage

``` r
is_ks_facts(x)
```

## Arguments

- x:

  An object.

## Value

`TRUE` if `x` is a `ks_facts`, otherwise `FALSE`.
