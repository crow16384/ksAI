# Capsule Tree as Nested List / Printable Structure

Capsule Tree as Nested List / Printable Structure

## Usage

``` r
capsule_tree(store, print = TRUE)
```

## Arguments

- store:

  A `ks_capsule_store`.

- print:

  Logical. If `TRUE`, print a tree to the console.

## Value

Invisibly, a named list of tree nodes.
